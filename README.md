# Event Management System (Java Backend)

Java backend system for managing events, users, registrations and sponsorships.

This project was originally developed as part of a distributed system including a web frontend (Servlets/JSP) and a mobile client. This repository focuses on the core backend logic and architecture.

## Features

- Object-oriented domain model (Users, Events, Registrations, Sponsorships)
- Layered architecture (Controllers, Business Logic, Persistence)
- Data Transfer Objects (DTOs)
- Custom exception handling
- Persistence layer using DAO pattern

## Architecture

- `logica/` → Core business logic and domain model
- `persistencia/` → Data access layer (DAO)
- `excepciones/` → Custom exceptions

## What I Learned

- Designing layered backend architectures
- Object-oriented modeling
- Separation of concerns
- Implementing controllers and business logic
- Working with persistence layers

## Notes

This backend was part of a larger system that also included:

- Web frontend using Java Servlets and JSP
- Desktop UI using Java Swing
- Mobile web client
- Communication via Web Services
